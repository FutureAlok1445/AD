1. Get Ready (Prerequisites) 🛠️
Install AWS CLI:

Run these commands if you haven't already:

Bash
sudo apt-get install curl
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
sudo apt install -y unzip
sudo unzip awscliv2.zip
sudo ./aws/install
aws --version 
Install Terraform:
Make sure Terraform is installed (from Exp 5 steps). Check with terraform -v.

Configure AWS Credentials:
Get your Access Key ID and Secret Access Key from the AWS Console (IAM -> My Security Credentials -> Create access key(Copy Access Key and Secret Key)).
Run: aws configure.
Enter your Key ID, Secret Key, default region (ap-south-1), and output format leave blank.

Create EC2 Key Pair in AWS but not launch EC2 instance:
Go to EC2 Console -> Key Pairs -> Create key pair.
Name it terraform (or match your variables.tf).
Choose .pem format and download the file securely.

2. Set Up Your Terraform Project 📁
mkdir project-terraform.
cd project-terraform.

3. Define Your Infrastructure (Write Code) 📝

####Create Variables File: sudo gedit variables.tf.
Paste this code (adjust default values if needed):

variable "aws_region" {
  description = "The AWS region to create things in."
  default     = "ap-south-1"  # Use your default region
}

variable "key_name" {
  description = "SSH keys to connect to ec2 instance"
  default     = "terraform"   # Must match the key pair name you created
}

variable "instance_type" {
  description = "instance type for ec2"
  default     = "t2.micro"    # Free-tier eligible instance type
}





####Create Main Configuration File: sudo gedit main.tf.
Paste this code:

provider "aws" {
  region = var.aws_region
}

#Create security group with firewall rules
resource "aws_security_group" "security_jenkins_port" {
  name        = "security_jenkins_port"
  description = "security group for jenkins"

  ingress {
    from_port   = 8080
    to_port     = 8080
    protocol    = "tcp"
    cidr_blocks = ["0.0.0.0/0"]
  }

  ingress {
    from_port   = 22
    to_port     = 22
    protocol    = "tcp"
    cidr_blocks = ["0.0.0.0/0"]
  }

  # outbound from jenkis server
  egress {
    from_port   = 0
    to_port     = 65535
    protocol    = "tcp"
    cidr_blocks = ["0.0.0.0/0"]
  }

  tags= {
    Name = "security_jenkins_port"
  }
}

resource "aws_instance" "myFirstInstance" {
  # USE A VALID AMI-ID FOR YOUR CHOSEN REGION
  ami           = "ami-_____" # Replace with your AMI ID
  key_name      = var.key_name
  instance_type = var.instance_type
  vpc_security_group_ids = [aws_security_group.security_jenkins_port.id]
  tags= {
    Name = "jenkins_instance"
  }
}

4. Run Terraform Commands 🚀
terraform init
terraform plan
terraform apply -> Action: Type yes and press Enter when prompted to confirm.

5.Verify Creation in AWS 👀
Go to AWS Console: Log in to your AWS account.
Check EC2: Navigate to the EC2 service.
Confirm: Look for:
A running instance named jenkins_instance.
A security group named security_jenkins_port.
An Elastic IP associated with the instance.

6.Destroy Infrastructure
terraform destroy -auto-approve
