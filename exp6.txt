1. Get Ready (Prerequisites) 🛠️
Install AWS CLI:

Run these commands if you haven't already:

Bash
sudo apt-get install curl
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
sudo apt install -y unzip
sudo unzip awscliv2.zip
sudo ./aws/install
aws --version 
Install Terraform:
Make sure Terraform is installed (from Exp 5 steps). Check with terraform -v.

Configure AWS Credentials:
Get your Access Key ID and Secret Access Key from the AWS Console (IAM -> My Security Credentials -> Create access key(Copy Access Key and Secret Key)).
Run: aws configure.
Enter your Key ID, Secret Key, default region (ap-south-1), and output format leave blank.

Create EC2 Key Pair in AWS but not launch EC2 instance:
Go to EC2 Console -> Key Pairs -> Create key pair.
Name it terraform (or match your variables.tf).
Choose .pem format and download the file securely.

2. Set Up Your Terraform Project 📁
mkdir project-terraform.
